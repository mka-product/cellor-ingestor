# Context Map

## Identity & Catalog

Owns slide identities, versions, upload completion, and ingestion job lifecycle.

## Ingestion

Consumes uploaded originals and produces derived artifacts, indexes, manifests, and provenance.

## Delivery

Owns immutable path generation, cache headers, and externally visible manifest references.

## Viewer

Consumes manifests and indexes, selects LOD, fetches tile groups, and renders tile bitmaps.
